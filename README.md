### **세상의 모든 줄임말을 담았다! 🤟**

온라인 세상 속 무수한 줄임말들 때문에 답답하셨죠?

JURIMMA는 여러분의 답답함을 해소시켜줄 수 있는 신세대 줄임말 단어 검색 서비스 입니다!

줄임말을 검색하고, 새로운 줄임말도 만들어보고, 퀴즈도 풀면서 줄임말 마스터가 되어보세요!
